Cómo ejecutar la aplicación

1. Abrir una terminal en la carpeta del proyecto.

2. Ejecutar el siguiente comando:

py -3.11 -m streamlit run app.py

3. Se abrirá una interfaz web en el navegador.

4. En la aplicación:
   - Subir una imagen del cielo
   - El sistema analizará la imagen
   - Si detecta una fase con confianza suficiente, devolverá:
     - creciente
     - llena
     - menguante
   - Si no alcanza el umbral de confianza, devolverá:
     - sin_luna

Requisitos:
- Python 3.11
- TensorFlow
- Pillow
- Streamlit

Archivos principales:
- entrenar.py → entrena el modelo de fases
- probar_modelo.py → prueba el modelo desde consola
- app.py → interfaz web para usar el modelo
- modelo_luna.h5 → modelo entrenado

Observación:
La lógica actual no entrena una clase explícita "sin_luna". Ese resultado se obtiene cuando ninguna de las tres fases supera el umbral de confianza definido.
